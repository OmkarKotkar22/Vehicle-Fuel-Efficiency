## ğŸ“Œ Project Overview  

The **Vehicle Fuel Efficiency** project is a **Core Java console application** that simulates how fuel efficiency of a vehicle can be calculated and displayed using object-oriented principles.  

Fuel efficiency is a crucial factor in the automotive industry, used to measure how far a vehicle can travel per unit of fuel. This project captures that real-life concept by defining a `Vehicle` model and a main execution class that processes and displays efficiency details.  

The project is intentionally designed to be **simple yet realistic**, helping beginners understand how Java classes interact in a real-world inspired scenario.
